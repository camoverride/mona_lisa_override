# Mona Lisa Override

## Setup

- `python -m venv .venv`
- `source .venv/bin/activate`
- `pip install -r requirements.txt`


## Run

-  `python run_display.py`


## Benchmark

- 20 secs to reach `print("Identifying faces")`, 20 secs more to finish -- not bad!
